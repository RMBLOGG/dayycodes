## ✨ Features

- 🎨 **Dark / Light mode** — tersimpan di localStorage
- 🌐 **Bilingual** — toggle Bahasa Indonesia / English
- 🤖 **Anime SVG character** — animasi rambut, mata, dan layar laptop
- 🚀 **10+ interaction animations** — ripple, particle burst, 3D tilt, spotlight, cursor trail
- 📊 **Scroll progress bar** — indikator scroll halaman
- 🗂️ **Project filter tabs** — slideable dengan animasi pill
- 📬 **Contact form** — dengan shake animation & flash message
- ⚡ **Particle canvas background** — efek bintang di hero section
- 📱 **Fully responsive** — desktop (16:9) & mobile

## 📁 Structure

```
portfolio/
├── app.py              # Flask app & project data
├── vercel.json         # Vercel deployment config
├── requirements.txt    # Python dependencies
├── templates/
│   └── index.html      # Main template (Jinja2)
└── static/
    ├── css/
    │   └── style.css   # Styles + animations
    └── js/
        └── main.js     # Interactions & effects
```

---

## 🚀 Run Locally

```bash
# Clone repo
git clone https://github.com/username/portfolio.git
cd portfolio

# Install dependencies
pip install -r requirements.txt

# Run
python app.py
```

Buka `http://127.0.0.1:5000` di browser.

---

## ☁️ Deploy ke Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Atau connect repo GitHub langsung di [vercel.com](https://vercel.com) → **New Project**.

---

## 🎨 Customization

Edit data proyek di `app.py` pada array `projects`:

```python
{
    "id": 1,
    "title": "Nama Proyek",
    "url": "https://link-proyek.vercel.app",
    "description": "Deskripsi singkat proyek.",
    "tags": ["Flask", "Python"],
    "badge": "Streaming",
    "accent": "#a78bfa",       # warna accent kartu
    "category": "streaming"    # all | streaming | api | web
}
```
